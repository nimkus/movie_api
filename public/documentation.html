Documentation: myFlix - Movie Web App

The “movies” web application provides users with access to information about different movies, directors, and genres. Users can to sign up, update their personal information, and create a list of their favorite movies.

Endpoints

Method Path Description
GET /movies Return a list of ALL movies to the user
GET /movies/:title Return data about a single movie by title (description, genre, director, image URL, whether it’s featured or not)
GET /movies/genres/:genreName Return data about a genre (description) by name/title (e.g., “Thriller”)
GET /movies/directors/:directorName Return data about a director (bio, birth year, death year) by name
POST /users/ Register new user
DELETE /users/:userId Deregister existing username (showing only a text that a user email has been removed)
PUT /users/:userId Update user info
PUT /users/:userId/:movieTitle Add a movie to list of user favorites (showing only a text that a movie has been added)
DELETE /users/:userId/:movieTitle Remove a movie from list of user favorites (showing only a text that a movie has been removed)
